**cafeiq**

Tech Stack: React 19 (TypeScript), Node.js/Express, MySQL, Socket.IO, Google Gemini AI, Vercel/Render Deployment

●	Built and deployed a full-stack Point of Sale (POS) system for coffee shops with QR-accessible digital menu, real-time order management, and AI-powered drink recommendations.
●	Integrated Google Gemini API for intelligent customer assistance—providing personalized drink suggestions and dietary-specific recommendations through an interactive chatbot interface.
●	Implemented real-time synchronization using Socket.IO—enabling live order updates, inventory tracking, and seamless communication between customer-facing interface and staff POS system.
●	Developed comprehensive loyalty rewards system with points earning/redemption, inventory management with automatic stock deduction, and multi-payment support (Cash, GCash, PayMaya) with receipt verification.
●	Deployed frontend to Vercel and backend to Render with automated CI/CD workflows—ensuring seamless updates and maintaining production stability across both platforms.
